Design Elements

  • Background Color: The background of these pages are brown to match the color I view traveling to be: the brown of a suitcase, or of a heavier coat. This carries through all the pages to create a cohesive website, and to create connections between pages.
  • Fonts: The font I chose to use for the headers of this site is called Akaya Kanadaka, which is a bendy font that gives the website a playful, yet informative feel. I chose to use Courier Prime for body text to create a contrast between the fun of traveling and the information about the country you might be traveling to. This font is strict and easy for anyone to read as well, which helps make this site accessible.
  • Font Color: I chose to use a darker red for the header colors to create a contrast between the background and the paragraph text. Red catches the attention of a viewer, and keeps it throughout our website. The paragraph text is a lighter red to create a hierarchy between the headers and the background, leading readers throughout the page.
  • Images: The circle images used on the other pages on this site help guide the viewers through the page. When they see the first circle images at the top of the page, they are led to read the rest of the text about this country below. The pictures to the side of the information about the countries are there to showcase the country the reader is learning about.
  • Repetition: The repetition utilized throughout this website can be seen in the header, decoration elements, images, and text. The layout on each page is very similar to the next, which guides the reader through each page with the same expectation. It takes the focus away from the website and pushes it toward the information on the page.
